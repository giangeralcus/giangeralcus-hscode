# HS Code Indonesia

Personal side project untuk mencari kode HS (Harmonized System) Indonesia berdasarkan BTKI 2022.

Tool sederhana untuk kegiatan customs clearance sehari-hari.

## Fitur

- Cari HS code berdasarkan nama barang
- Database 11,552 kode HS 8-digit Indonesia (E-BTKI 2022)
- 11,194 kode dengan tarif (BM MFN, PPN)
- Deskripsi bahasa Indonesia & Inggris
- History pencarian tersimpan
- Dark mode UI

## Data Sources

| Source | Description |
|--------|-------------|
| [BTKI 2022](https://www.beacukai.go.id/btki.html) | Buku Tarif Kepabeanan Indonesia 2022 |
| [INSW](https://www.insw.go.id/) | Indonesia National Single Window |
| [E-BTKI 2022](https://www.beacukai.go.id/) | Electronic BTKI v2.1 April 2022 |
| [UN Comtrade](https://comtradeplus.un.org/) | International HS codes |
